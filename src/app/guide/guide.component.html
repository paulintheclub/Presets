Usage Instructions Usage Instructions
For Lightroom Mobile (DNG File Format) Download Preset Begin by downloading the DNG file containing the VividSnap preset to your mobile device. Import to Lightroom Mobile Open Adobe Lightroom Mobile and create a new album. In this album, tap the '+' sign and choose 'Add Photos' to import the DNG file. Create Preset Open the DNG file in the album, tap on the three dots in the upper right corner, and select 'Create Preset'. Name Your Preset Give your preset a name, for example, 'VividSnap', and save it in a preset group of your choice for easy access. Apply Preset To apply the preset, simply navigate to the photo you wish to edit, scroll to 'Presets' at the bottom, and select 'VividSnap' from your list of presets.
For Lightroom Desktop (XMP File Format) Download Preset Save the XMP file to your computer. Open Lightroom Launch Adobe Lightroom Desktop. Install Preset Go to the 'Develop' module, find the 'Presets' panel on the left side, click on the '+' icon, and select 'Import Presets'. Navigate and Select Find where you saved the XMP file on your computer, select it, and click 'Import'. Apply Preset The VividSnap preset will now appear in your 'Presets' panel. To use it, just click on the preset name when you're in the 'Develop' module editing a photo.
